Phidget Plugin for vvvv (beta 1.0)


To Install the plugin in vvvv:

- Download drivers 2.1.6 from www.phidgets.com and install.  
- copy the compiled files phidgetInterface.dll into plugin folder


You can find all information about phidgets on
http://www.phidget.com
